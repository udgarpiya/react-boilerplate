# react-boilerplate
React with Webpack, Babel, ESLint, Jest.

## To Run This Project

- First, install all needed dependencies

```sh
npm i
```

- Then, start the server

```sh
npm run dev
```

## To Build This Project

```sh
npm run build
```

## To Run Test

```sh
npm run test
```

## To Run Test coverage

```sh
npm run test:coverage
```

## To Run lint

```sh
npm run lint
```
